## Description

In this task, I built a Convolutional Neural Network (CNN) model to classify handwritten digits using the MNIST dataset. The MNIST dataset contains 70,000 images of handwritten digits from 0 to 9. Each image is 28x28 pixels in size.

I used TensorFlow and Keras libraries to build the CNN model. The model has two Convolutional layers followed by MaxPooling layers to extract important features from the images. After that, a Flatten layer and two Dense layers are used to classify the digits.

The model was trained for 5 epochs on 60,000 training images and tested on 10,000 test images. The model achieved a test accuracy of 98.9 percent which means it correctly identified almost all handwritten digits.

I also plotted accuracy and loss graphs which clearly show that the model learned well over each epoch without overfitting. Sample predictions were also made and all 5 predictions were 100 percent correct.

This task helped me understand how CNN models work, how images are processed and classified using deep learning, and how to evaluate model performance using accuracy and loss graphs.

Tools used in this task are Python, TensorFlow, Keras, NumPy, Matplotlib, and Google Colab.

The output of this task is a Jupyter Notebook with complete CNN model, accuracy graphs, and digit predictions.
